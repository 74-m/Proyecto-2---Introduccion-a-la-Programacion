# Proyecto-2---Introduccion-a-la-Programacion
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Instrucciones de ejecuicón

Para ejecutar este programa se requiere:

1.Tener instalado Python 3.12.

2.Instalar Pygame.

3.Descargar todos los archivos y mantenerlos dentro de una misma carpeta, incluye los archivos de recursos, ya que son necesarios para el correcto funcionamiento del programa.

Una vez que está seguro de que tiene todo lo anterior puede ejecutar el programa sin problemas, si desea conocer más sobre el funcionamiento y el manual del juego, puede revisar la documentación adjunta.
